This app shows some of my wildlife sightings. Upon clicking on a marker or on an animal in the list to the left, information from Wikipedia and related images on Flickr are retrieved and shown in this info panel. You can search through the animals using the search filter. MVC is incorporated using knockoutJS. This is an ongoing project, future goals consist in adding a database and the option to add more animals to the map.
